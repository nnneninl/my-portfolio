# 🌐 My Portfolio Website

Welcome to my personal portfolio website! This site showcases my work, skills, and projects. You can view the live demo of my portfolio website [here](https://neilfrancispalmares.com)

## 💻 Tech Stack

- Astro 🚀 - Static site generation framework.
- React ⚛️ - For dynamic components and interactivity
- Tailwind CSS 🌊 - Utility-first CSS framework for styling.
- shadcn/ui ✨ - Pre-styled UI components for rapid development.
- AWS ☁️ - Deployed and hosted on Amazon Web Services for reliable cloud hosting.

## 📜 Features

- A responsive layout that adapts to both desktop and mobile views 📱💻.
- Dark and light theme support 🌙☀️.
- Integrated with React for interactive components 🎮.
- Continuous deployment workflow to AWS with GitHub Actions 🔄.

## 🏁 Getting Started

To run and view the project locally, follow these steps:

1. Clone the Repository

```bash
git clone https://github.com/nnneninl/my-portfolio.git
cd my-portfolio
```

2. Install Dependencies

```bash
npm install
```

3. Run the Development Server

```bash
npm run dev
```

The site should now be running at http://localhost:4321.
