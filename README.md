# 🚀 My Website Portfolio

This is my personal website portfolio, build with **Next.js**, **Tailwind CSS**, **Framer Motion**, **shadcn/ui**, and **react-icons**.

🌐 Live Demo: [https://neilfancis.vercel.app](https://neilfrancis.vercel.app)

## 🔥 Tech Stack

- **Next.js** - React framework for server-side rendering and routing.
- **Tailwind CSS** - Utility-first styling framework for fast and consistent design.
- **Framer Motion** - Animation library for smooth and engaging interactions.
- **shadcn/ui** - Pre-styled UI components for quickl and elegant design.
- **react-icons** - Icon library with customizable and scalable icons.

## 🌟 Features

✅ Responsive and mobile-friendly layout  
✅ Smooth animations with Framer Motion  
✅ Dark and Light Mode support  
✅ Fast performance optimized Next.js build
